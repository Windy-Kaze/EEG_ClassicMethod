## 安装

```bash
git clone https://github.com/Windy-Kaze/EEG_ClassicMethod.git
cd your-repo
pip install -r requirements.txt
```

## 使用方法
在main.py文件展示了一次经典的使用流程。

## 示例
详见main.py文件。一次经典的流程为
```
-1.初始化数据集对象
-2.配置基本和附加信息(实例方法添加)
-3.加载数据
-4.配置并应用滤波器(实例方法添加)
-5.创建算法对象并训练/测试
```
